# Go Gin Next.js Auth

Features:
- [x] JWT (Refresh + Access Tokens)
- [x] RBAC (user, admin)
- [x] Secure password hashing (bcrypt)
- [x] Google OAuth2
- [x] Token revocation (logout)
- [x] Email verification
- [ ] Refresh token rotation
- [ ] Token blacklist or versioning
- [ ] Password reset
- [ ] Account lock after N failed logins
- [ ] Rate Limiter (per-ip, per-user)
- [ ] Change password
- [ ] Unit tests (password hash, token validation, permission checks)
